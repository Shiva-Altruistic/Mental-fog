## 🎯 Problem Statement

Many students and professionals experience mental fog, lack of clarity, and difficulty focusing on tasks, but lack a simple tool to guide structured thinking.

## 💡 Solution

Clarity is an AI-powered assistant that helps users:
- Express thoughts
- Identify mental blocks
- Get actionable steps to regain focus


# 🧠 Clarity – Mental Fog Assistant

An AI-powered assistant designed to help users overcome mental fog, regain focus, and improve productivity through intelligent conversation.



## ✨ Features

* 🧠 Detects signs of mental fog through conversation
* 💬 Interactive chatbot interface
* ⚡ Provides actionable suggestions for focus and clarity
* 🔄 Context-aware responses using AI
* 🌐 Full-stack architecture (Frontend + Backend)

---

## 🛠️ Tech Stack

**Frontend**

* React (TypeScript)
* Vite

**Backend**

* Node.js
* Express

**AI Integration**

* Vertex AI (via proxy API)

## ⚙️ Installation & Setup

###  Clone the repository

```
git clone https://github.com/Shiva-Altruistic/Mental-fog.git
cd clarity-ai
```

---

###  Install dependencies

#### Root (optional)

```
npm install
```

#### Backend

```
cd backend
npm install
```

#### Frontend

```
cd ../frontend
npm install
```

---



### Run the application

#### Start backend

```
cd backend
node server.js
```

#### Start frontend

```
cd frontend
npm run dev
```
